# カート獲得ツール（C: 喪失通知 / D: ダッシュボード）

## D. カート獲得ダッシュボード
**Amazon出品管理 → 商品サマリータブ** の上部に表示。
- **カート獲得率**・○獲得/△新品優先/✗未獲得の件数
- **🎯 あと一歩でカート獲得**: ✗のうちカート価格との差が小さい順。「差額」だけ値下げすれば獲得が狙える
- **📈 獲得率の推移**: 毎時更新で「カート履歴」タブに蓄積され折れ線表示
- **🔄 今すぐ更新**ボタン: その場で最新サマリーを取得（手動更新）

データは商品サマリーの更新（毎時自動 or 各種「更新」ボタン）で更新される。

## C. カート喪失 Discord 通知（任意設定）
サマリー更新時に「前回○ → 今回○でない（カート喪失）」を検知し、Discordに通知する。
**Webhook URL を設定したときだけ**動作する（未設定なら静かにスキップ）。

### 設定手順
1. Discord でチャンネルの **連携サービス → ウェブフック → 新しいウェブフック** を作成し、URLをコピー
2. 通知を動かしたい場所に応じて URL を登録:
   - **毎時の自動更新（GitHub Actions）で通知** → リポジトリの Secrets に
     `DISCORD_WEBHOOK_URL` を追加（`gh secret set DISCORD_WEBHOOK_URL` でも可）
   - **アプリ内の更新ボタンで通知** → Streamlit Cloud の Secrets に
     ```toml
     [discord]
     webhook_url = "https://discord.com/api/webhooks/..."
     ```
3. 以降、サマリー更新でカートを落とした商品があれば Discord に一覧が届く

通知例:
```
⚠️ カート喪失アラート（2 件）
・SONY ウォークマン A20シリーズ  販売¥9,870 / カート¥9,500  (SKU ...)
・ニンテンドー3DS コスモブラック  販売¥13,980 / カート¥12,580  (SKU ...)
```
